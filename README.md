# **Portfolio**

## **Installation**

To install Portfolio, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/ant-vu/ant-vu.github.io.git
```

2. Install dependencies:

```bash
npm install
```

3. Start the project:

```bash
npm start
```
